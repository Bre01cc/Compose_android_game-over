# Compose_android_game-over
## Sobre
Projeto desenvolvido para colocar em prática os conhecimentos adquiridos da aula de jatpack compose da disciplina mobile lecionada por [João henrique](https://www.linkedin.com/in/joao-henrique-parizoti-meyer/?locale=pt_BR).

## Descrição
Aplicação é uma "landing page" de uma interface de game over de um jogo, onde foram usados conceitos de alinhamento e mudança de elemento(cor, tamanho, formato).

## Ferramentas
- Android Studio
- Kotlin
